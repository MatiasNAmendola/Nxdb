Nxdb.Persistence.Attributes.PersistentCollectionAttribute Class Reference

Stores or fetches a collection to/from a child element of the container element. If more than one element with the given name exists, the first one will be used. This supports arrays, lists, etc. The persistent object must either be an array, implement ICollection<T>, or implement IList and provide an ItemType. By default this stores and fetches data in the following format: 

 <Container>
   <Name>
     <Item>...</Item>
     <Item>...</Item>
     <Item>...</Item>
   </Name>
 </Container>

Detailed Description

Stores or fetches a collection to/from a child element of the container element. If more than one element with the given name exists, the first one will be used. This supports arrays, lists, etc. The persistent object must either be an array, implement ICollection<T>, or implement IList and provide an ItemType. By default this stores and fetches data in the following format: 

 <Container>
   <Name>
     <Item>...</Item>
     <Item>...</Item>
     <Item>...</Item>
   </Name>
 </Container>
