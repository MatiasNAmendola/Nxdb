IEnumerable<T> Nxdb.Persistence.Manager.GetObjects< T > ( Element  parent,
string  expression,
bool  attach,
bool  attachItems 
)

Returns a collection of objects created by evaluating the specified query against the specified parent element and attaches it to the parent element if specified. This requires the specified object type to provide an accessible empty constructor. If one is not available, an exception will be thrown.

Template Parameters:
TThe type of objects to construct.
Parameters:
parentThe element against which to evaluate the expression to get result elements to be used for the construction of new objects.
expressionThe expression to evaluate. Only Element results will be used (I.e., no text elements, attributes elements, etc. will be used).
attachIf set to true attaches the newly created collection, otherwise just populates it.
attachItemsIf set to true attaches result objects, otherwise does not attach result objects.
Returns:
A collection of persistent objects of the specified type. Though the return object can be explicitly refreshed and is also automatically refreshed (if enabled for this manager), explicitly saving or deleting it has no effect.
